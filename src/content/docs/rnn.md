Recurrent Neural Networks: Temporal Learning

Core Concept

· Sequence Data Processing: Time series, text, speech
· Memory Mechanism: Hidden state retains past information
· Parameter Sharing: Same weights across time steps

Basic RNN Architecture

````
Input → [RNN Cell] → Hidden State → Output
         ↑    ↓
             Previous Hidden State
             ```

 ---

             Mathematical Foundations of RNN

             Forward Propagation Equations

             Time Step Hidden State Output Update Gate*
             t=0 $h_0 = \text{tanh}(W_{xh}x_0 + W_{hh}h_{-1} + b_h)$ $y_0 = W_{hy}h_0 + b_y$
             t=1 $h_1 = \text{tanh}(W_{xh}x_1 + W_{hh}h_0 + b_h)$ $y_1 = W_{hy}h_1 + b_y$
             t=2 $h_2 = \text{tanh}(W_{xh}x_2 + W_{hh}h_1 + b_h)$ $y_2 = W_{hy}h_2 + b_y$
             ... ... ...

             *For LSTM/GRU only

             ---

             Deep RNN: Stacked Architecture

             Multi-Layer RNN Structure

             ```
             Input → [RNN Layer 1] → Hidden₁ → [RNN Layer 2] → Hidden₂ → [RNN Layer N] → Output
                        ↓                   ↓                     ↓
                                State₁              State₂                Stateₙ
                                ```

                                Advantages

                                · Hierarchical Feature Extraction: Different layers capture different temporal abstractions
                                · Increased Modeling Capacity: More parameters, complex patterns
                                · Better Gradient Flow: Through multiple pathways

                                ---

                                Deep RNN Mathematical Formulation

                                Layer-wise Forward Pass

                                Layer 1: $h_t^1 = \text{tanh}(W_{xh}^1 x_t + W_{hh}^1 h_{t-1}^1 + b_h^1)$

                                Layer 2: $h_t^2 = \text{tanh}(W_{xh}^2 h_t^1 + W_{hh}^2 h_{t-1}^2 + b_h^2)$

                                Layer L: $h_t^L = \text{tanh}(W_{xh}^L h_t^{L-1} + W_{hh}^L h_{t-1}^L + b_h^L)$

                                Output: $y_t = W_{hy} h_t^L + b_y$

                                ---

                                Weight Update Table: Backpropagation Through Time (BPTT)

                                Fill in the weight updates for each iteration:

                                Parameter Initialization Update Rule (Gradient) Iteration 1 Iteration 2 Iteration 3
                                $W_{xh}$ Random small values $\frac{\partial L}{\partial W_{xh}} = \sum_{t} \frac{\partial L}{\partial h_t} \cdot \frac{\partial h_t}{\partial W_{xh}}$
                                $W_{hh}$ Random small values $\frac{\partial L}{\partial W_{hh}} = \sum_{t} \frac{\partial L}{\partial h_t} \cdot \frac{\partial h_t}{\partial W_{hh}}$
                                $b_h$ Zeros $\frac{\partial L}{\partial b_h} = \sum_{t} \frac{\partial L}{\partial h_t} \cdot \frac{\partial h_t}{\partial b_h}$
                                $W_{hy}$ Random small values $\frac{\partial L}{\partial W_{hy}} = \sum_{t} \frac{\partial L}{\partial y_t} \cdot \frac{\partial y_t}{\partial W_{hy}}$
                                $b_y$ Zeros $\frac{\partial L}{\partial b_y} = \sum_{t} \frac{\partial L}{\partial y_t}$

                                ---

                                Python Implementation: RNN from Scratch

                                ```python
                                import numpy as np

                                class RNN:
                                    def __init__(self, input_size, hidden_size, output_size):
                                            # Weight initialization
                                                    self.Wxh = np.random.randn(hidden_size, input_size) * 0.01
                                                            self.Whh = np.random.randn(hidden_size, hidden_size) * 0.01
                                                                    self.Why = np.random.randn(output_size, hidden_size) * 0.01
                                                                            self.bh = np.zeros((hidden_size, 1))
                                                                                    self.by = np.zeros((output_size, 1))

                                                                                                def forward(self, inputs):
                                                                                                        h = np.zeros((self.Whh.shape[0], 1))
                                                                                                                self.inputs = inputs
                                                                                                                        self.hidden_states = { -1: h.copy() }
                                                                                                                                self.outputs = []

                                                                                                                                                for t, x in enumerate(inputs):
                                                                                                                                                            h = np.tanh(self.Wxh @ x + self.Whh @ h + self.bh)
                                                                                                                                                                        y = self.Why @ h + self.by
                                                                                                                                                                                    self.hidden_states[t] = h.copy()
                                                                                                                                                                                                self.outputs.append(y)
                                                                                                                                                                                                        return self.outputs, h

                                                                                                                                                                                                                def backward(self, dL_dy, learning_rate=0.01):
                                                                                                                                                                                                                        # Initialize gradients
                                                                                                                                                                                                                                dWxh, dWhh, dWhy = np.zeros_like(self.Wxh), np.zeros_like(self.Whh), np.zeros_like(self.Why)
                                                                                                                                                                                                                                        dbh, dby = np.zeros_like(self.bh), np.zeros_like(self.by)
                                                                                                                                                                                                                                                dh_next = np.zeros_like(self.hidden_states[0])

                                                                                                                                                                                                                                                                # Backpropagation through time
                                                                                                                                                                                                                                                                        for t in reversed(range(len(self.inputs))):
                                                                                                                                                                                                                                                                                    # Output layer gradients
                                                                                                                                                                                                                                                                                                dy = dL_dy[t]
                                                                                                                                                                                                                                                                                                            dWhy += dy @ self.hidden_states[t].T
                                                                                                                                                                                                                                                                                                                        dby += dy

                                                                                                                                                                                                                                                                                                                                                # Hidden state gradients (through output and next hidden state)
                                                                                                                                                                                                                                                                                                                                                            dh = self.Why.T @ dy + dh_next

                                                                                                                                                                                                                                                                                                                                                                                    # Backprop through tanh
                                                                                                                                                                                                                                                                                                                                                                                                dh_raw = (1 - self.hidden_states[t]**2) * dh

                                                                                                                                                                                                                                                                                                                                                                                                                        # Update parameter gradients
                                                                                                                                                                                                                                                                                                                                                                                                                                    dbh += dh_raw
                                                                                                                                                                                                                                                                                                                                                                                                                                                dWxh += dh_raw @ self.inputs[t].T
                                                                                                                                                                                                                                                                                                                                                                                                                                                            if t > 0:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                            dWhh += dh_raw @ self.hidden_states[t-1].T

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    dh_next = self.Whh.T @ dh_raw

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    # Update weights
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            self.Wxh -= learning_rate * dWxh
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    self.Whh -= learning_rate * dWhh
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            self.Why -= learning_rate * dWhy
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    self.bh -= learning_rate * dbh
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            self.by -= learning_rate * dby
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            ```

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                prev_size = input_size

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                for hidden_size in hidden_sizes:

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        prev_size = hidden_size

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        self.Why = np.random.randn(output_size, prev_size) * 0.01
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                self.by = np.zeros((output_size, 1))

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        def forward(self, inputs):
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                layer_input = inputs
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        hidden_states_per_layer = []

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        for layer in self.layers:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    layer_output, hidden_states = layer.forward(layer_input)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                hidden_states_per_layer.append(hidden_states)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            layer_input = layer_output

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            # Final output
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    outputs = [self.Why @ h + self.by for h in layer_input]
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            return outputs, hidden_states_per_layer
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            ```

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            Training Deep RNN: Challenges & Solutions

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            Vanishing/Exploding Gradients

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            · Problem: Gradients diminish/explode through deep layers and time steps
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            · Solutions:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              · Gradient clipping
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                · Weight initialization (Xavier/He)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  · Gated architectures (LSTM, GRU)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    · Skip connections

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    Training Techniques

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    1. Teacher Forcing: Feed ground truth as next input during training
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    2. Curriculum Learning: Simple to complex sequences
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    3. Gradient Checkpointing: Memory optimization for BPTT
