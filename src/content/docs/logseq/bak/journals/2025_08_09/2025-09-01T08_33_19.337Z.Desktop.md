3. Pin & Organize

Open dashboard.md in one tab.

Use VS Code Split View: left = dashboard, right = whatever you’re working on.

Pin dashboard tab so it’s always there.

Assign keyboard shortcut → Ctrl+Shift+D → “Open Dashboard” command.

---

4. Calendar Integration

In Outlook:

1. Go to Settings → Calendar → Shared Calendars → Publish a Calendar.

2. Copy the ICS link.

3. In VS Code Calendar View extension, add the ICS feed.
   Now your events appear right inside VS Code.

---

5. Task Integration

Install vscode-microsoft-todo.

Sign in with Microsoft account.

Choose “Today” list to display.

You can add tasks directly from VS Code with /todo command.

GitHub Actions (optional): every night, commit your Foam workspace to backup.

Got it — you want a lean, cross-platform productivity setup that plays nicely with VS Code, Foam, Marp, Microsoft To Do, and Outlook, without bloating your workflow.

Here’s a complementary minimal stack that fills the missing pieces while staying portable:

---

1. Notes & Knowledge Management

You already have Foam (Markdown in VS Code) — keep it as your “second brain.”

Sync: Use Git + GitHub/GitLab to version & sync notes across devices.

Quick capture on mobile:

Obsidian Mobile (reads the same Markdown repo as Foam)

Or GitJournal (Android/iOS, commits to Git)

---

2. Presentations

You have VS Code + Marp for Markdown-based slides.

Add Deckset (Mac) or Slidev (cross-platform) if you ever want fancier, code-driven presentations without switching tools.

Store presentations alongside notes in the same Git repo.

---

3. Tasks & Planning

You’re using Microsoft To Do — good for Outlook integration.

Tip: Set up keyboard shortcuts for quick task entry.

For offline + Git-based backup, mirror your key tasks in a simple tasks.md file in your Foam workspace.

---

4. Calendar & Scheduling

Outlook Calendar (integrated with To Do & email) — stick with it.

For a minimal weekly view dashboard inside VS Code, try CalVA or Calender View extension (pulls from ICS feeds).

---

5. Writing & Content

VS Code + Markdown for all writing.

Typora or MarkText as clean distraction-free editors if you want a “focus mode.”

---

6. Mind Mapping / Visual Thinking

Excalidraw (web, desktop, VS Code plugin) — works offline, free-form drawing + embeds in Markdown.

Store .excalidraw files in the same Foam repo.

---

7. Automation & Capture

Raycast (Mac) / Flow Launcher (Windows) — universal quick-launcher.

Espanso — cross-platform text expansion (snippets, templates).

---

8. Mobile & Cross-Device Sync

Use OneDrive (since you’re in Microsoft ecosystem) or Syncthing if you prefer local sync without cloud lock-in.

For Markdown-based systems, keep everything in one repo → clone to all devices → works offline.

---

9. Minimal Distraction Layer

Cold Turkey (Windows) / SelfControl (Mac) — block distracting sites/apps.

Or simply use Focus Mode in VS Code + hide activity bar.
